## Overview

Quanta is a developer-focused, open-source programming knowledge-sharing platform inspired by modern technical communities.

It provides a structured and scalable frontend architecture for:

- Developer Q&A discussions  
- Technical blog publishing  
- Community forums & groups  
- User profile & achievements  
- Help center & ticket support system  

This project is designed for extensibility and future backend integration.

---

## Why Quanta?

- Modular and enterprise-ready architecture  
- Clean separation of concerns  
- Scalable component design  
- Contributor-friendly structure  
- Built using modern frontend best practices  

Quanta aims to become a fully open-source programming community ecosystem.

## Project Structure

```bash
frontend/
│
├── public/
│ ├── assets/
│ └── vendor/
│
├── src/
│ ├── components/
│ ├── context/
│ ├── data/
│ ├── pages/
│ └── assets/
│
└── configuration files

Architecture principles:

- `components/` → Reusable UI building blocks  
- `pages/` → Route-level screens  
- `context/` → Global state management  
- `data/` → Mock and static data  
```

## Installation

Clone the repository:

```bash
git clone https://github.com/rohankumarrawat/Quanta-Website.git
cd Quanta-Website/frontend
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

## Contribution Guidelines

- Follow existing folder structure
- Keep components modular
- Write clean and readable code
- Follow Tailwind design patterns
- Ensure no linting errors

For major changes, please open an issue first to discuss what you would like to change.
