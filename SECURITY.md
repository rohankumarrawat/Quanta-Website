# Security Policy

## Supported Versions

We currently support the latest stable version of Quanta.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

---

## Reporting a Vulnerability

If you discover a security vulnerability:

- Do NOT open a public issue.
- Email the maintainer directly (add your contact email here).
- Provide detailed steps to reproduce the issue.

We will:

- Acknowledge receipt within 48 hours
- Investigate promptly
- Release a fix if necessary
- Credit responsible disclosure

---

## Security Best Practices

- Keep dependencies up to date
- Avoid exposing sensitive data
- Follow secure coding standards
- Review third-party libraries carefully

We appreciate responsible security reporting.
